# Agnos WebSocket Server

Real-time WebSocket server for patient-staff communication and form management system.

## Installation

### Prerequisites

- Node.js >= 18

### Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

For development with auto-reload:
```bash
npm run dev
```

### Environment Variables

- `PORT` - Server port (default: 4000)
